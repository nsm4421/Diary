# Diary App

본 애플리케이션(이하 “앱”)은 이용자의 개인정보를 보호하기 위해 최선을 다하고 있으며, 본 개인정보 처리방침은 앱 이용 과정에서 개인정보가 어떻게 처리되는지 설명합니다. 앱을 이용함으로써 본 개인정보 처리방침에 동의하는 것으로 간주됩니다.

앱은 사용자의 개인 식별 정보(이름, 이메일, 전화번호 등)를 직접 수집하지 않으며, 위치 정보, 주소록, 사진 및 미디어 파일, 광고 ID, 기기 ID 등 추적 관련 정보도 수집하지 않습니다. 단, 앱 이용 과정에서 Google Play 등 운영체제나 스토어 시스템이 오류 로그나 기본 사용 데이터를 자동으로 수집할 수 있으나, 앱은 해당 데이터에 접근하거나 저장하지 않습니다.

앱은 사용자 정보를 식별하거나 추적하기 위한 목적으로 어떤 정보도 이용하지 않으며, 앱이 정상적으로 동작하는 데 필요한 최소한의 기능만 수행합니다. 또한 앱은 사용자 정보를 외부에 제공하거나 제3자와 공유하지 않습니다. 앱은 개인정보를 저장하지 않기 때문에 별도의 개인정보 삭제 절차는 필요하지 않습니다. 앱을 삭제하면 기기에 저장된 앱 관련 데이터는 모두 자동으로 삭제됩니다.

이 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다. 만약 아동의 정보가 수집된 것으로 판단되는 경우 아래 연락처로 알려주시면 확인 후 즉시 삭제 조치를 취하겠습니다.

본 개인정보 처리방침은 법령 변경, 서비스 업데이트 등의 필요에 따라 수정될 수 있으며, 변경 시 최종 수정일을 갱신하여 공지합니다. 개인정보 처리와 관련된 문의 사항은 언제든지 nsm4421@naver.com
